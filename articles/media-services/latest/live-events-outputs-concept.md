# Live Events and Live Outputs

Azure Media Services enables you to deliver live events to your customers on the Azure cloud. To configure your live streaming events in Media Services v3, you need to learn about [Live Events](https://docs.microsoft.com/rest/api/media/liveevents) and [Live Outputs](https://docs.microsoft.com/rest/api/media/liveoutputs).
